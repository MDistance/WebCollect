## ejs 的使用步骤
1. 安装
```html
npm install ejs
```

2. express 中不需要手动导入
```html
//设置express使用的模板引擎
app.set('view engine', 'ejs');
//设置模板的存放位置
app.set('views', __dirname + '/views');// 视图 模板
```

3. 在回调中 解析模板
```html
app.get('/admin', function(req, res){
    res.render('admin');
});
```
